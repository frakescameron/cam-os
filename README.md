💻 Cam-OS — Custom Web-Based Operating System

Cam-OS is a browser-based operating system built from scratch using modern web technologies. It simulates a desktop environment with applications, navigation, and system-like interactions — all running in the browser.

🚀 Overview

Cam-OS is designed to replicate the feel of a lightweight operating system while remaining fast and accessible through a web interface.

Key characteristics:

Desktop-style UI with windows, apps, and navigation
Lightweight performance without heavy frameworks
Interactive environment that feels like a real system


🧠 Purpose

This project focuses on building deeper technical understanding by:

Simulating OS-level concepts in a web environment
Practicing front-end architecture and state management
Creating a scalable system for adding custom applications
Building something that goes beyond a typical website


🛠️ Tech Stack
React / JSX — application structure and UI components
JavaScript — core logic and interactivity
CSS — layout, styling, and desktop simulation
Vite — fast development and build tooling
GitHub Pages — deployment


🧩 Features
🖥️ Desktop Interface — windowed layout with app-style navigation
📁 Projects System — organized section for accessing work
🎮 Terminal Integration — launch hidden apps like secretgame
⚡ Fast Load Times — minimal overhead, optimized builds
🎨 Custom UI/UX — built from scratch, no prebuilt OS frameworks


🎯 Design Philosophy
Keep it fast and responsive
Keep it modular and expandable
Keep control over every system component
Avoid unnecessary abstraction
